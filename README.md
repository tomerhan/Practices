## Algorithms

Common algorithm implementations and exercises:

- **Sorting**: Bubble Sort, Merge Sort, Quick Sort
- **Searching**: Binary Search, Linear Search
- **Recursion**: Fibonacci, Factorial, Tree Traversal
- **Dynamic Programming**: Memoization, Tabulation

See [`algorithms/`](./algorithms/) for examples.

## Data Structures

Foundational data structure implementations:

- **Linear**: Arrays, Linked Lists, Stacks, Queues
- **Tree-based**: Binary Trees, Heaps
- **Hash-based**: Hash Maps, Sets

See [`data-structures/`](./data-structures/) for examples.

## Design Patterns

Common software design patterns with examples:

- **Creational**: Singleton, Factory, Builder
- **Structural**: Adapter, Decorator, Facade
- **Behavioral**: Observer, Strategy, Command

See [`design-patterns/`](./design-patterns/) for examples.

## Clean Code Practices

Guidelines and examples for writing clean, maintainable code:

- Meaningful names
- Small, focused functions
- Avoiding side effects
- Writing self-documenting code

See [`clean-code/`](./clean-code/) for examples.

## Getting Started

Each folder contains standalone examples in Python. Run any example directly:

```bash
python algorithms/sorting/merge_sort.py
python data-structures/linked_list.py
```
